Configures the font and column properties for the Treeview widget.

Details:
    - Sets up two columns (`#1` and `#2`) in the Treeview widget.
    - Specifies the width of column `#1` as 150 units.
    - Specifies the width of column `#2` as 250 units.

str anchor ) 
Packs the Treeview widget into a specific position within its parent frame.

Args:
    - side (str): The side of the frame to place the widget. 
        Possible values are "TOP", "LEFT", "RIGHT", or "BOTTOM".
    - padx (int | tuple): Horizontal padding around the widget.
    - pady (int | tuple): Vertical padding around the widget.
    - anchor (str): Alignment of the widget relative to the packing side 
        (e.g., "n", "s", "e", "w", "center", etc.). 
        (this parameter is not used in the current implementation)

@author : Lakhdar Gibril

◆ set_text()

None TreeviewBuilder.TreeviewBuilder.set_text ( self,
str text ) 
Sets text values for the column headers of the Treeview widget.

Args:
    - text (str): The text value to add to the first column header 
        (currently not directly used in the method).

Details:
    - Sets the heading for column `#0` with the text 'Generated Image'.
    - Sets the heading for column `#1` with the text 'Image name'.
    - Sets the heading for column `#2` with the text 'Simulation Name'.
    - All headings are center-aligned.
